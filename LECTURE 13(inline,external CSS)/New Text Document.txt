INLINE CSS:-

In inline css ham html k syntax k ander hee css ko use krte hein. eg <p style="color: red;">This tutorial will teach you.....</p>
style = color : red; inline css hei baki <p> tag html tag h to write paragraph..We can also write more properties in a single style as <p style="color:red; background-color:yellow;"> etc

INTERNAL CSS:-

In internal css we write it in the head body by inserting first style tag and then write everything in the style tag. eg <style> p{color:red}</style> p is for paragraph and it is known as selector. But if we have applied both inline and internal css both the visual studio or anyother will always give preference to inline css rather than internal css (do whatever written in inline css) , but if we remove inline css then it will do according to internal css.
